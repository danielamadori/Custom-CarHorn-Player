# Car Horn ESP32-WROOM32D
SW and HW for customization of a car horn.

This version integrates the [MusicPlayer](https://github.com/eivindlie/MusicPlayer) library
to handle melodies defined by notes and durations. A Python parser for MusicXML
is provided under `Software/lib/MusicXMLParser` to help converting scores.
