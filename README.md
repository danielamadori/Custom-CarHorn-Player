# Car Horn ESP32-WROOM32D
SW and HW for customization of a car horn.

This project now includes the [NonBlockingRTTTL](https://github.com/end2endzone/NonBlockingRTTTL)
library to play melodies defined in RTTTL format. The player can parse notes and
durations from a song string and reproduce it using the horn pin.
